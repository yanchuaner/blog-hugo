# 润色阶段 Prompt

润色以下技术文章。保持内容不变，仅优化表达。

## 规则

1. 保持第一人称（"我"），保持开发者对话感
2. 中英文之间加空格（正确：DeepSeek API；错误：DeepSeekAPI）
3. 中文使用全角标点（，。！？）
4. 代码块内注释用中文
5. 拆分过长的段落（超过 5 行），但不要拆分代码块
6. 技术术语使用标准写法
   - C++ 不是 cpp
   - JavaScript 不是 js（除非在代码中）
   - GitHub 不是 github
7. 消除口语化表达（"这个东西"→"这个特性"）
8. 消除冗余修饰（"非常"、"特别"、"真的很"）
9. 确保每个"为什么"都有对应的"怎么做"
10. 不要改变代码块内容
