# 博客优化 Todo List (第三阶段：UI视觉与灵动感提升)

> 前期基础架构、性能和安全优化已全面完成。
> 本阶段核心目标：围绕**“灵动感 (Dynamic & Alive)”**与**“现代视觉质感 (Modern Aesthetics)”**，打造顶级开发者 Portfolio 体验。

---

## 🔴 高优先级（P0）— 核心微交互与动效

### 1. 英雄区 (Hero) 交错进场动画
- **目标**：首屏加载时告别死板的整体展现，带来富有生命力的视觉冲击。
- **方案**：
  - 将 `.spa-hero-grid` 的单一 FadeUp 拆解为交错动画 (Staggered Animation)。
  - `Badge` (标签) -> `Name` (姓名) -> `Role` (头衔) -> `Bio` (简介) -> `Actions` (按钮) 依次以 100ms~150ms 的延迟从下方滑入并淡入。
  - 右侧头像框引入极其缓慢的上下浮动动画 (Floating effect)。
- **涉及文件**：`assets/css/spa.css`

### 2. 卡片探照灯/流光悬浮特效
- **目标**：打破传统的 `translateY` 悬浮，引入现代 Web3 / Vercel 风格的交互质感。
- **方案**：
  - 为 `Project` 和 `Skill` 卡片添加监听鼠标位置的探照灯特效 (Mouse-tracking glow) 或渐变边框流光。
  - 当鼠标在卡片上移动时，卡片背景或边框局部会跟随鼠标位置产生微弱的高亮反光。
- **涉及文件**：`assets/css/spa.css`, `assets/js/spa.js`

### 3. 引入平滑页面转场 (Page Transitions)
- **目标**：解决传统多页静态站点击跳转时白屏或生硬切页的问题。
- **方案**：
  - 尝试启用浏览器原生的 `View Transitions API` (`@view-transition`)。
  - 对于不支持的浏览器，使用简单的拦截点击 + 页面淡出 (Fade out) 后跳转，以及页面加载时的全局淡入 (Fade in)。
- **涉及文件**：`layouts/_partials/extend_head.html`, `assets/js/theme-init.js`

## 🟡 中优先级（P1）— 质感与动态背景

### 4. 智能悬浮导航栏 (Smart Sticky Header)
- **目标**：提升阅读沉浸感，同时保持导航便利。
- **方案**：
  - 监听滚动方向：向下滚动时导航栏平滑上滑隐藏，为阅读腾出空间；向上滚动时立刻平滑下滑出现。
  - 优化 Header 的毛玻璃效果，使其在滚动时透明度和模糊度显得更加“通透” (Advanced Glassmorphism)。
- **涉及文件**：`assets/css/spa.css`, `assets/js/theme-footer.js`

### 5. 背景粒子互动增强
- **目标**：让背景“活”起来，与用户产生共鸣。
- **方案**：
  - 修改 `spa.js` 中的 Canvas 粒子引擎，记录鼠标的 `(x, y)` 坐标。
  - 使粒子在靠近鼠标指针时，产生轻微的排斥效果 (Repel) 或连线增强发光效果，提升交互乐趣。
- **涉及文件**：`assets/js/spa.js`

## 🟢 低优先级（P2）— 细节打磨与高级排版

### 6. 流光文字与强调色
- **目标**：增加页面重点信息的视觉表现力。
- **方案**：
  - 将 Hero 区域的姓名或核心能力标签替换为动态渐变色 (`background-clip: text` + 慢速背景平移缓动动画)。
  - 对代码块背景或区块边框引入极其细微的渐变映射。
- **涉及文件**：`assets/css/spa.css`

### 7. 定制化现代滚动条
- **目标**：消除浏览器默认滚动条的突兀感。
- **方案**：
  - 使用 `::-webkit-scrollbar` 将滚动条变窄（如 `6px`），滑块使用半透明的圆角设计，并在 hover 时加深颜色，完美融入暗黑/高亮主题。
- **涉及文件**：`assets/css/spa.css`
